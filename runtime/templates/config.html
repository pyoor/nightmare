Configuration

Working path: Path where all the crashing samples will be stored for later analysis.
Temporary path: Path where temporary files, packets, etc... will be created.
Nightmare path: Path where Nightmare Fuzzing Project is installed.
Queue host: Hostname or IP address of the (Beanstalk) queue server.
Queue port: Port where the (Beanstalk) queue server is listening.
